Go to the https://github.com/abettername/MyMidterm.git and clone (or fork) the repository “MyMidterm”.

Open Terminal, move to MyMidterm directory and create a .txt file “MyName”. Type in it your First and Last name. Make sure it is saved in your MyMidterm folder.

In your Browser open your MyName.txt file and copy the text, then through any text Editor or VS open the html file named “DesignMe.html” from “MyMidterm”, and paste your name instead of “REPLACE THAT”. Save it. Open “DesignMe.html” from a Browser.

In any text editor or VS Edit “DesignMe.html” to make your name style a better fit to your personality. Save it. Open it in a Browser to check the results. Make a screenshot and save your restyled name in “MyMidterm” folder as MyName in portable network graphics format.

Open “MyCertificate.fig” in Figma, insert “MyName.png” into the file and make changes to the design to ensure that your name is the focal point of the layout, correct the esthetic mistakes in the formatting. Export it as “MyExamCertificate” in pdf format to your “MyMidterm”.

Create a separate minimalistic version of your certificate for mobile devices: Open “MobileCertificate.html” in any text editor or VS, finish and debug the code. Name it “MyMidtemExam.html” and push it into your github repository.
